The Tape

Each game takes place on a tape 12 to 32 cells long. The first (and last) cell is the register. The second (and second to last) cell is the flag. Each cell can hold a value from 0 to 255, which wraps around. All cells start with a value of zero at the beginning of the game, except for the flags, which start at 128.
